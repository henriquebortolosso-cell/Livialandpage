Coloque o vídeo da demonstração nesta pasta com o nome:

    video.mp4

Ele será reproduzido automaticamente quando alguém clicar no botão
"play" da seção de estatísticas (o círculo verde com triângulo).

Formatos suportados pelo navegador: MP4 (H.264) é o mais compatível.
Tamanho recomendado: até 50 MB para carregar rápido.

Se preferir outro nome de arquivo, edite a linha
    video: "video/video.mp4"
no objeto CONFIG dentro de index.html.
